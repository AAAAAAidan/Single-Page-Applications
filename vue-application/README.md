# Vue Application

A simple application for practice with Vue. This project was bootstrapped with [Vue CLI](https://cli.vuejs.org/guide).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
